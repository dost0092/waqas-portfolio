About

I am a Data Scientist and Data Engineer with 1 years of experience at Zoneomics, specializing in geospatial and land use data. At Zoneomics, I managed complex zoning, land use and permits datasets, designing robust data pipelines to collect and process city permit information. Using Python and SQL, I built ETL workflows that integrate GIS spatial data into PostgreSQL databases. I developed RESTful APIs for client data delivery and created interactive dashboards (e.g. a Cities Dashboard and a Client & User Monitoring Dashboard) using Retool, Tableau and Power BI to visualize key metrics and trends.

In addition, I handle cloud deployment and DevOps tasks on AWS (Elastic Beanstalk, S3), containerizing applications with Docker. I maintain CI/CD pipelines using GitHub and Jenkins to ensure reliable, automated delivery of data services. I hold relevant certifications (AWS Cloud Practitioner Essentials and multiple DataCamp certificates) and leverage tools like Python, Tableau, and Power BI to turn raw data into actionable insights. I am detail-oriented, results-driven, and committed to using data-driven solutions to address complex challenges.
