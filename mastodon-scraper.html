Mastodon Social Explore Page Scraper

This project is a fully automated Python-based scraper built to extract multimedia (images, videos, audio) and textual content from the Mastodon Social Explore page. It navigates dynamically loaded content using Selenium, performs structured data extraction, and saves the results in organized local folders and CSV files for downstream use in data analysis, content monitoring, or dataset creation.

  • Objectives:
    • Extract images, videos, audio files, and text posts from Mastodon's public Explore feed.
    • Automate infinite scrolling to dynamically load more content.
    • Save extracted media to local folders with clean, deduplicated filenames.
    • Store text and media URLs in structured .csv files.
    • Enable reuse for data mining, content curation, or research purposes.
  • Technical Details:
    • Image Scraping: Uses Selenium to scroll the page and detect <img> elements. Filters valid image URLs (JPEG, PNG), downloads and stores them locally in /image folder, handles filename parsing and uniqueness.
    • Audio Scraping: Captures <audio> tags dynamically loaded on the Explore page, extracts .mp3, .ogg, or streamable audio URLs, downloads to /Audio directory with error handling.
    • Video Scraping: Identifies <video> tags and collects .mp4 or video stream links, scrolls through page to ensure all content is loaded, saves video links to mastodon_videos.csv.
    • Text Content Extraction: Leverages BeautifulSoup with headless Chrome (via Selenium), parses loaded HTML content, filters meaningful human-readable text, saves content in mastodon_text.csv with proper encoding and cleanup.
  • Output Structure:

  • Key Features:
    • Auto Scrolling: Smoothly scrolls until all content is loaded.
    • Intelligent Filtering: Ensures only usable media/text content is saved.
    • Data Preservation: Downloads assets with original names when possible.
    • Structured Outputs: CSV and folder-based layout for further analysis.
  • Use Cases:
    • Media analysis from social platforms
    • Dataset building for computer vision or audio modeling
    • Social content archiving
    • Content moderation or trend research
    • Real-time scraping base for AI-powered agents
