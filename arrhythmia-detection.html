Exploring Arrhythmia Detection Through ECG Data Analysis: A Comprehensive Study

This project explores arrhythmia detection using large-scale 12-lead ECG data, leveraging advanced preprocessing, visualization, and dimensionality reduction techniques to uncover patterns and support early diagnosis.

  • Dataset: WFDBRecords - 12-lead ECG recordings for arrhythmia study
  • Objectives:
    • Acquire, preprocess, and analyze ECG data
    • Store and retrieve ECG signals and metadata in PostgreSQL
    • Visualize ECG signals and attributes
    • Apply PCA for dimensionality reduction and pattern discovery
    • Support early diagnosis and management of arrhythmias
  • Technical Steps:
    • Data acquisition with wfdb and psycopg2
    • Recursive directory traversal and data cleaning
    • Storing ECG and metadata as JSON in PostgreSQL
    • Visualization with matplotlib and widgets
    • PCA for dimensionality reduction and 3D visualization
  • Code Highlights:

...
# Visualization and PCA

  • Output:
    • Cleaned, structured ECG data in PostgreSQL
    • Interactive visualizations of ECG signals and attributes
    • PCA-transformed 2D/3D plots for pattern discovery
  • Key Features:
    • Handles large-scale ECG datasets
    • Automated data cleaning and storage
    • Comprehensive visualization of cardiac signals
    • Dimensionality reduction for insight discovery
  • Technologies Used:
    • Python, wfdb, psycopg2, PostgreSQL, matplotlib, scikit-learn, numpy, ipywidgets
