Building a Custom API Integration in Microsoft Word with VBA and UserForm

This project demonstrates how to build a Word macro in VBA that connects to an external API using a custom UserForm for user input. The macro securely stores API keys, sends requests, parses JSON responses, and inserts results directly into the Word document.

  • Objectives:
    • Enable real-time API data integration in Word via VBA.
    • Gather user input with a custom UserForm.
    • Store and manage API keys securely in document properties.
    • Send API requests and handle responses in VBA.
    • Parse JSON and insert results into the document.
  • Technical Steps:
    • Enable Developer tab and set up VBA environment.
    • Add Microsoft Scripting Runtime and XML, v6.0 references.
    • Store API key in custom document property.
    • Create UserForm for user input (coordinates, address, etc.).
    • Build FetchData subroutine to send API requests.
    • Parse JSON responses and insert data into Word.
    • Add error handling for API and JSON issues.
