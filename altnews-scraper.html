Altnews Website Scraper

This project is a Python-based scraping tool designed to extract content from Altnews.in, a leading Indian platform focused on fact-checking and misinformation debunking. The scraper gathers images, audio files, text content, and video links from the site, enabling researchers or data analysts to build structured datasets for content verification, media monitoring, or disinformation studies.

  • Objectives:
    • Extract visual, audio, video, and textual content from the Altnews homepage or article pages.
    • Handle modern HTML structures using Selenium and BeautifulSoup.
    • Store data in organized folders and CSV files for downstream NLP or ML usage.
    • Offer a reusable framework for monitoring other news or fact-checking websites.
  • Technical Details:
    • Image Scraping: Navigates the homepage or articles using Selenium. Collects <img> tags and filters valid http image URLs. Downloads all media into a local /image folder using requests and PIL.
    • Audio Scraping: Uses BeautifulSoup to parse the page and extract links (<a href>) pointing to .mp3, .ogg, .wav. Handles audio file downloads into a separate /Audio directory. Uses urljoin() to ensure valid absolute URLs even when relative links are found.
    • Text Content Extraction: Collects all human-readable content from the HTML DOM using soup.find_all(text=True). Excludes scripts and styles. Saves text content in a clean, tab-separated format (altnews_text.csv) inside a /text directory.
    • Video Link Extraction: Uses Selenium to scroll dynamically loaded pages. Collects <video> elements and their src attributes. Saves links in a altnews_videos.csv file under the /videos folder for future download or review.
  • Output Structure: 
    altnews-scraper/
│
├── image/                      # Downloaded images
├── Audio/                      # Audio files
├── text/
│   └── altnews_text.csv        # Extracted readable text
├── videos/
│   └── altnews_videos.csv      # Extracted video URLs
├── image_scraper.py
├── audio_scraper.py
├── text_scraper.py
└── video_scraper.py
  • Key Features:
    • Dynamic Content Loading: Handles infinite scroll for deeper content discovery.
    • Absolute URL Resolution: Converts all relative paths using urljoin.
    • Clean Data Output: Filters noisy HTML and extracts human-readable data only.
    • File Management: Ensures properly named and deduplicated media files.
  • Use Cases:
    • Archiving and analyzing misinformation correction efforts
    • Creating a training dataset for fake news classification models
    • Monitoring changes in fact-checking content across time
    • Comparing media patterns across fact-checking vs mainstream sites
